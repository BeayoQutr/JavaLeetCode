# Java LeetCode 180 天系统刷题

这是一个面向 Java 学习者的 LeetCode 系统训练仓库，提供完整的 180 天刷题路线，并按训练日组织 Java 代码。目标是循序渐进地掌握常见数据结构与算法，形成稳定的分析、编码和复盘习惯。

## 路线特点

- 180 个编号训练日，每天安排 2 道题，共 360 次训练任务。
- 覆盖 307 道不同题目，并通过复习日重复训练代表题。
- 内容从数组、哈希、字符串开始，逐步覆盖链表、树、图、回溯、贪心和动态规划等主题。
- 同时关注算法思路、时间复杂度、空间复杂度、边界条件和间隔复习。

## 仓库结构

```text
.
├── README.md
├── LEETCODE_ROADMAP_180_DAYS.md
└── src/
    └── dayXX/
        └── Number<题号>_Solution<解法编号>.java
```

- [`LEETCODE_ROADMAP_180_DAYS.md`](./LEETCODE_ROADMAP_180_DAYS.md)：完整的 180 天训练路线与每日题单。
- [`src/`](./src)：按训练日组织的 Java 代码。

## 使用方法

1. 打开刷题路线，选择当前 Day 的两道题。
2. 先独立分析暴力思路，再推导更优方案。
3. 使用 Java 完成代码并测试空输入、单元素、重复值和极值等边界。
4. 记录核心思路与复杂度，并按路线中的复习节奏重新实现错题。

## 环境要求

- JDK 17 或更高兼容版本
- 任意支持 Java 的 IDE 或编辑器


建议从 [`180 天系统刷题路线`](./LEETCODE_ROADMAP_180_DAYS.md) 开始使用本仓库。
